# 1.2 Justificativa

O Pong é um dos primeiros jogos icônicos e uma referência importante na história dos jogos. Atualizá-lo para um formato multijogador via navegador é uma forma de homenagear esse clássico enquanto explora as capacidades modernas da web. A implementação com Python (Django) e Golang para o Backend e Typescript com React para o Frontend oferece uma solução robusta para comunicação em tempo real e interatividade.

A inclusão de uma IA e a possibilidade de torneios tornam o jogo mais envolvente e diversificado, proporcionando uma experiência de jogo rica e desafiadora que pode ser compartilhada e jogada com amigos em microcomputadores. A adoção de uma arquitetura de microsserviços permitirá uma maior flexibilidade e eficiência na gestão dos diferentes componentes do sistema.
